Table Code

(table) tells the browser that the information inside will be a table.

(thead) stands for table header. It includes the titles of each column.

(tbody) stands for table body. It includes the rows of data.

(tfoot) stands for table footer. It includes the last row of data. We typically do not use tfoot unless you want some special styling on the last row.

(th) stands for table header cell. It is the name of a column at the top of a table.

(tr) stands for table row. It represents a single row in the table

(td) stands for table data. It represents a single cell or piece of data

Example:

Surname Name Grade GPA
Joel Joe 9th 2.993
Garcia Eduardo 10th 3.55
